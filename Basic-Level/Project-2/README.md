### Deployment Stages

We will deploy the WordPress application in the following stages:

1. **Localhost / Server-based deployment**

   * Understand traditional deployment fundamentals
   * Learn how application components interact at the OS level

2. **Docker-based deployment**

   * Containerize the WordPress application
   * Run and manage services using containers

3. **Kubernetes (K8s) cluster deployment**

   * Deploy the same application in a Kubernetes environment
   * Learn scalability, service management, and orchestration concepts

Each stage builds on the previous one, reinforcing the idea that **tools change, but deployment principles remain the same**.
